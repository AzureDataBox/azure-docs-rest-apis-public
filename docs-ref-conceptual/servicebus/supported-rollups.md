# Supported Rollups
## Supported Rollups  
 Telemetry and usage data include the following metrics:  
  
|Metric|Applicable Entities|  
|------------|-------------------------|  
|**PT5M**|5-minute rollup|  
|**PT1H**|1-hour rollup|  
|**P1D**|1-day rollup|  
|**P7D**|1-week (7 days) rollup|
